# Full Stack Blog Platform

## Overview

The Full Stack Blog Platform is a web application that allows users to create, edit, delete, and manage blog posts. The project was developed using the MERN Stack and provides a simple and user-friendly interface for blog management.

## Features

* Create new blog posts
* Edit existing blog posts
* Delete blog posts
* View all blogs
* Responsive and user-friendly interface

## Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB
* JavaScript
* HTML5
* CSS3

## Project Objective

The objective of this project is to develop a full-stack web application and gain practical experience in frontend development, backend development, API integration, and database management using the MERN Stack.

## Learning Outcomes

* Developed responsive user interfaces using React.js
* Built REST APIs using Node.js and Express.js
* Connected applications with MongoDB database
* Improved problem-solving and full-stack development skills
